# Web Page - Plantilla de Empresa

Plantilla simple para una página corporativa. Está pensada para editarse fácilmente: cambia el logo, el nombre de la empresa y los textos.

Ediciones rápidas:

- Actualiza el nombre y el logo en `index.html` (busca "Nombre de la Empresa" y `you logo.png`).
- Cambia el correo en el formulario de contacto en `index.html` (`contacto@empresa.com`).
- Ajusta colores y tipografías en `styles.css`.

Uso:

- Abrir `index.html` en un navegador para ver la página.
- Para probar en local con recarga, sirve la carpeta con un servidor estático (ej. `npx http-server` o extensiones de Live Server en VS Code).

¡Personaliza el contenido y pon en marcha tu web!

# English

Web Page - Company Template

Simple template for a corporate website. It's designed to be easy to edit: change the logo, the company name and the texts.

Quick edits:

- Update the name and logo in `index.html` (look for "Nombre de la Empresa" and `your logo.png`).
- Change the email in the contact form in `index.html` (`contacto@empresa.com`).
- Adjust colors and fonts in `styles.css`.

Usage:

- Open `index.html` in a browser to view the page.
- To test locally with live reload, serve the folder with a static server (e.g. `npx http-server` or the Live Server extension in VS Code).

Customize the content and launch your site!
